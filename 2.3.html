En este último subtema, te darás cuenta que otra forma de identificar los modelos de evaluación de un proyecto educativo, además del FODA, existen los Elementos de un Proyecto, a través de las características de los Modelos de Evaluación de Proyectos Sociales; divididos en:

  1. a) Modelo analítico
  2. b) Modelo global
  3. c) Modelo Formalizado
  4. d) Modelo semi – formalizado.

Estos modelos de evaluación de proyectos, está enfocado desde la perspectiva social que hemos estado revisando desde inicio de la asignatura, dado que el área educativa es un acto social.

Podrás identificar en la relación de este proyecto que cada uno de estos tipos de modelos son un conjunto de aplicaciones y procedimientos a partir del cual es posible la implementación de un proceso de evaluación de un determinado proyecto. Son esquemas o diseños teóricos, metodológicos y operativos, así como instrumentos y técnicas de recolección de datos que permiten la evaluación, desde otra perspectiva, del propio proyecto educativo.

A continuación, se hará una pequeña descripción de cuál es la característica de este modelo de evaluación en proyectos sociales.
